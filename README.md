🚀 CareerCraft – AI Mock Interview Agent
🎯 CareerCraft is an AI-powered mock interview platform designed to help candidates practice interviews, improve communication skills, and receive intelligent feedback on their answers.

The system simulates real interview scenarios and evaluates responses using modern AI techniques such as Natural Language Processing (NLP) and Retrieval-Augmented Generation (RAG).

Whether you are preparing for technical interviews, behavioral questions, or general HR rounds, CareerCraft helps you practice in a realistic and structured way.

✨ Features
🧠 AI Interviewer
Simulates real interview questions and interacts with candidates like a real interviewer.

📊 Answer Evaluation
Analyzes candidate responses based on correctness, relevance, and completeness.

📚 Knowledge Retrieval (RAG)
Retrieves correct answers or key concepts from a knowledge base before evaluating responses.

💬 Natural Conversations
Uses NLP to understand answers even if the wording differs from the expected answer.

📈 Feedback & Scoring
Provides structured feedback and scores to help candidates understand where they can improve.

🧠 How It Works
CareerCraft combines AI and information retrieval to simulate interview experiences.

1️⃣ Question Generation
The system asks interview questions based on predefined datasets or topics.

2️⃣ Candidate Response
The candidate answers the question through text input.

3️⃣ Retrieval-Augmented Generation (RAG)
Instead of relying only on the AI model’s internal knowledge, the system retrieves relevant information from a knowledge base or database.

4️⃣ Answer Evaluation
The AI compares the candidate’s answer with the expected answer and evaluates:

correctness

relevance

completeness

5️⃣ Feedback Generation
The system generates constructive feedback to help candidates improve their answers.
